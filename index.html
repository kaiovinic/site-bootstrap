Bootstrap

O que é Bootstrap?

É um framework de código aberto, desenvolvido pelo Twitter, que fornece um conjunto de ferramentas e recursos para facilitar o desenvolvimento web responsivo e a criação de interfaces de usuário consistentes e atraentes. Ele é baseado em HTML, CSS e JavaScript e oferece uma variedade de componentes, como grids, tipografia, formulários, botões, navegação e outros elementos de interface.

Bootstrap via Link (CDN):

Facilidade de implementação: Basta adicionar um link para o Bootstrap no cabeçalho do seu HTML e começar a usar imediatamente.

Atualizações automáticas: Como você está referenciando uma versão hospedada externamente do Bootstrap, quaisquer atualizações ou correções de bugs serão aplicadas automaticamente à sua página.

Bootstrap via Download e inclusão local:

Maior controle: Você tem controle total sobre a versão do Bootstrap que está usando e pode decidir quando atualizá-la.

Independência de rede: Não há dependência de um CDN externo, então seu site não será afetado se o CDN estiver inacessível.
